# Anjali Portfolio – Full Project

- Home with resume-style sections (Summary → Education → Experience → Projects → Skills → Certifications)
- Professional header with photo + contacts, Download Resume, Print/PDF, Dark mode
- Dedicated `/certifications` page with badge cards + modal
- Tailwind ready, Router wired
- Netlify + GitHub Pages configs included

## Run
npm install
npm run dev

## Deploy
- GitHub Pages: `.github/workflows/github-pages.yml` builds on push to main
- Netlify: `netlify.toml` (build `npm run build`, publish `dist`)

## Assets
Put files in `/public`:
- `profile.jpg`
- `Anjali_Gunjale_Resume.pdf`
- `certifications/*`
